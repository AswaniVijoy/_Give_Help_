
# 💝 GiveHelp – Crowdfunding Platform

A **full-stack crowdfunding web application** where users can explore fundraising campaigns and donate to meaningful causes, while administrators manage campaigns through an admin dashboard.

The platform provides a simple way to track **campaign goals, raised amounts, and donation progress**.

---

# 🚀 Features

### 👤 User

* User registration and login
* Browse crowdfunding campaigns
* View campaign details
* Donate to campaigns
* Track donation progress
* Update profile

### 🛠️ Admin

* Admin authentication
* Create campaigns
* Edit campaigns
* Delete campaigns
* View donations
* Manage campaign status
